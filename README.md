### Getting Started
```
# 1. clone the repo
git clone git@github.com:hectorredx/GTAV-Tunables.git

# 2. install dependencies
npm install

# 3. start the script
npm start

# 4. Checkout the decrypted tunables saved as "tunables.json" in the output directory
```
